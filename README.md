# manufacturing-quality-analysis-dashboard

Built an automated reporting system using SQL, Python, and Power BI to generate KPI-driven dashboards and business insights.

This project is about analyzing manufacturing data to understand production performance and identify defects using SQL and Power BI.

## Project Overview
In this project, I worked with manufacturing data to find useful insights like defect rates, production trends, and areas where improvements can be made. The goal was to turn raw data into meaningful information that can help in better decision-making.

## Tools Used
- SQL  
- Power BI  
- Excel  

## What I Did
- Cleaned and prepared the dataset using Excel  
- Wrote SQL queries to analyze production and defect data  
- Built an interactive dashboard in Power BI  
- Created KPIs like total production, total defects, and defect rate  
- Visualized trends to understand performance over time  

## Files Included
- `manufacturing_data.xlsx` – Dataset used  
- `queries.sql` – SQL queries  
- `dashboard.pbix` – Power BI dashboard  

## Outcome
This project helped in understanding how data can be used to identify issues in manufacturing and improve overall efficiency.
